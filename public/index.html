Available Endpoints

GET /api/health

Health check endpoint - returns server status

GET /api/hello

Basic hello message with timestamp

GET /api/users

Get all users

POST /api/users

Create a new user (requires name and email in request body)

GET /api/users/{id}

Get a specific user by ID

GET /api/external?source={source}

Proxy external API calls. Available sources: jsonplaceholder, httpbin, quotes

Example Usage

Create a new user:

curl -X POST /api/users \
  -H "Content-Type: application/json" \
  -d '{"name":"John Doe","email":"john@example.com"}'

Get external API data:

curl "/api/external?source=quotes"
